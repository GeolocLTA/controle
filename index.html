Ferramenta de controle de rotas - CDR

Bem vindo a Ferramenta CdR (Controle de Rotas), projeto acadêmico da disciplina PI 2 em Engenharia da Computação da turma EC5MA do CESUPA.
Tem como objetivo fazer o gerenciamento da empresa ao seus automóveis, sejam eles qual forem, usando este site desenvolvido usando linguagens como HTML, CSS e JavaScript para demonstrar graficamente as rotas geradas pelos dispositivos usados e portanto buscando essas informações enviadas ao Firebase, que está integrado a este site.
